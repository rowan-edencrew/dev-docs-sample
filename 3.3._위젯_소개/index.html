위젯이란¶

위젯(Widget)은 Lucy Studio에서 사용자 인터페이스를 구성하는 기본 요소 입니다.

텍스트, 이미지, 버튼, 화면 레이아웃까지 모두 위젯으로 간주됩니다.

위젯을 페이지에 적절히 배치하고 속성을 설정하여 앱의 시각적 디자인과 기능을 완성합니다.

위젯 생성 및 배치¶

위젯을 캔버스에 직접 배치하거나 **Widget Tree**를 활용하여 기존 구조 내에 삽입할 수 있습니다.

위젯의 생성 및 배치 방법은 다음과 같습니다.

1. 캔버스에 드래그 앤 드롭¶

Widgets 패널에서 원하는 위젯을 선택한 뒤, 캔버스에 드래그 앤 드롭하여 추가합니다.

2. Widget Tree에서 Layout의 자식으로 추가¶

1) Widget Tree에서 Row, Column과 같은 Layout을 선택하고 우클릭

2) Add a child to this widset 메뉴 클릭

3) Floating Widget Plate에서 해당 레이아웃의 자식 위젯으로 새로운 위젯 추가

3. Widget Tree에서 Widget의 앞/뒤에 삽입¶

1) Widget Tree에서 특정 위젯을 선택한 후 우클릭

2) Insert widset before this widset 또는 Insert widset after this widset 메뉴 클릭

3) Floating Widget Plate에서 선택한 위젯 앞이나 뒤에 새로운 위젯 추가

위젯 카테고리¶

Lucy Studio의 위젯은 기능과 역할에 따라 다음과 같이 네 가지 주요 카테고리로 분류됩니다.

카테고리 주요 역할 주요 위젯 (예시)
레이아웃 위젯 다른 위젯들을 담거나 **배치하는 구조**를 정의합니다. Row, Column, Stack, Wrap, GridView 등
구분 위젯 UI 요소 간의 **공간을 확보**하거나 **경계를 구분**하는 데 사용됩니다. Spacer, Divider, Vertical Divider, SpaceBox 등
콘텐츠 위젯 **정보를 표시**하거나 **사용자의 입력**을 받는 상호작용 요소입니다. Text, Image, Button, TextField, Icon 등
금융 위젯 금융 서비스나 특정 산업에 특화된 **고급/복합 기능을 수행**하는 위젯입니다. SimpleLineChart, SimpleBarChart, StockChart 등
