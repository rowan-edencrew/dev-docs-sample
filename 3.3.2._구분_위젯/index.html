구분 위젯이란¶

구분 위젯은 요소들 사이의 간격을 조절하거나 시각적인 경계선을 추가하여 레이아웃의 완성도를 높이는 역할을 합니다.

구분위젯에는 Spacer, Divider-H, Divider-V, SpaceBox가 있습니다.

구분 위젯 종류¶

위젯 설명 사용 팁
Spacer Row, Column 등 위젯에서 남은 여유 공간을 채워줍니다. 툴바 좌우 정렬 간격
Divider-H 레이아웃 요소를 구분하기 위한 수평선입니다. 섹션 헤더 아래 얇은 선
Divider-V Row 등 위젯 내에 수직선으로 콘텐츠 영역을 구분합니다. 버튼 그룹 수직 분리
SpaceBox 고정된 크기의 빈 박스를 삽입하여 레이아웃 요소 간격을 명시적으로 조정합니다. 고정 간격, 카드 간 패딩 대용

Spacer¶

Spacer 위젯은 Row, Column과 같은 Flex 레이아웃 내에서 자식 위젯들 사이의 **남은 공간을 차지**하여 간격을 만듭니다.

1. flex 속성¶

  • 남은 공간을 차지할 **비율**을 지정합니다. 여러 Spacer가 있을 경우 설정된 비율에 따라 공간을 나누어 가집니다.
    • 모든 Spacer의 flex 값이 같다면, 남은 공간을 공평하게 나누어 가집니다.
    • 특정 Spacer의 flex 값을 높이면, 그 비율만큼 더 많은 공간을 차지하게 됩니다.
    • flex를 0으로 설정하면 공간을 차지하지 않습니다.

2. flex 설정에 따른 Spacer의 크기¶

  • 남은 공간이 ‘60’, Spacer 가 2개 있을 경우

    flex 값 Spacer 크기
    1 : 1 30 : 30
    1 : 3 15 : 45
    1 : 0 60 : 0

이처럼 flex 속성을 활용하면 다양한 화면 크기에 유연하게 대응하는 반응형 레이아웃을 손쉽게 구현할 수 있습니다.

Divider-H와 Divider-V¶

Divider-H와 Divider-V는 콘텐츠 영역을 시각적으로 구분하기 위해 수평 또는 수직의 실선을 그리는 위젯입니다.

선이 차지하는 전체 영역과 실제 선의 두께를 개별적으로 제어하여 시각적인 효과를 줄 수 있습니다.

1. 개요 및 차이점¶

  • Divider-H: 수평 방향의 구분선을 렌더링하며, 주로 Column 내에서 리스트 항목이나 섹션을 나눌 때 사용합니다.
  • Divider-V: 수직 방향의 구분선을 렌더링하며, Row**나 **Flex 컨테이너 내에서 콘텐츠를 좌우로 분리할 때 사용합니다.
**구분 위젯 ‘`Divider-H` / `Divider-V`’ 와 레이아웃 위젯 ‘`VSplit` / `HSplit`’ 의 차이** 모두 화면을 나눈다는 공통점이 있지만, **'단순히 시각적 구분선을 긋는가'**와 **'영역을 구조적으로 분할하는가'**라는 핵심적인 차이가 있습니다. | **구분** | **Divider-H / Divider-V** | **VSplit / HSplit** | | --- | --- | --- | | **카테고리** | 구분 위젯 (Divider) | 레이아웃 위젯 (Layout) | | **핵심 역할** | 시각적 경계선 표시 (선 긋기) | 공간의 논리적 분할 (틀 짜기) | | **시각적 요소** | 실질적인 '선'을 렌더링 | 영역을 구조적으로 분할 | | **예시** | 목록 아이템 사이의 얇은 회색 선 | 대시보드의 메뉴/본문 분할 |

2. 속성(Property) 정리¶

속성 타입 기본값 설명
visible bool true false 설정 시 구분선을 렌더링하지 않으며 레이아웃에서 제외됩니다.
height / width double 0.0 **Divider-H**는 수직 공간(height), **Divider-V**는 수평 공간(width)을 지정합니다. 이는 여백을 포함한 전체 점유 영역입니다.
thickness double 0.0 화면에 실제로 그려지는 선의 순수한 두께입니다. 0.0일 때 기본값 1px 이 적용됩니다.
indent double 0.0 선의 시작 부분(Divider: 왼쪽, VerticalDivider: 위쪽) 들여쓰기 값을 지정합니다.
endIndent double 0.0 선의 끝 부분(Divider: 오른쪽, VerticalDivider: 아래쪽) 들여쓰기 값을 지정합니다.
color Color 테마 기본값 선의 색상을 지정합니다. 커스텀 색상이나 시스템 테마 색상을 선택할 수 있습니다.

3. 세부 설명 및 활용 팁¶

  • 공간(Height/Width) vs 선 두께(Thickness)
    • height(또는 width)는 구분선이 차지하는 **전체 투명한 박스 영역**의 크기입니다.
    • 실제 눈에 보이는 선의 굵기는 오직 thickness 속성으로만 조절합니다.
    • 예시: height: 20, thickness: 1 설정 시, 20px의 투명한 공간 중앙에 1px 두께의 선이 그려집니다.
  • 들여쓰기(Indent)를 활용한 디자인
    • indent와 endIndent를 활용하면 전체 너비(또는 높이)를 다 채우지 않는 세련된 구분선을 만들 수 있습니다.
    • Divider-H: indent는 왼쪽, endIndent는 오른쪽 여백을 조절하여 선의 길이를 짧게 만듭니다.
    • Divider-Vr: indent는 위쪽, endIndent는 아래쪽 여백을 조절하여 선의 길이를 짧게 만듭니다.

SpaceBox¶

SpaceBox는 고정된 크기의 빈 공간을 삽입할 때 사용합니다.

Spacer가 유동적인 비율로 공간을 채운다면, SpaceBox는 명확한 픽셀 단위의 간격을 줄 때 유용합니다.

1. 속성(Property) 정리¶

속성 타입 기본값 설명
visible bool true 공간의 가시성 여부를 결정합니다. false 시 공간 자체가 사라집니다.
size.width double / ∞ ∞ 공간의 너비를 지정합니다. ∞ 설정 시 가로 방향의 남은 공간을 모두 채웁니다.
size.height double / ∞ ∞ 공간의 높이를 지정합니다. ∞ 설정 시 세로 방향의 남은 공간을 모두 채웁니다.
연동(Lock) icon locked 너비와 높이 값을 동일하게 연동하여 변경할지 여부를 선택합니다. (Lock 클릭 시 연동)

2. 세부 설명 및 활용 팁¶

  • size.width & size.height
    • 고정 크기: 숫자 입력 시 정확한 픽셀 크기의 공백을 생성합니다.
    • 유동 크기: ∞ 클릭 시 해당 축의 남은 공간을 모두 차지하는 Spacer처럼 동작합니다.
  • 잠금(연동)
    • locked: width 변경 시 height가 자동으로 동일 값으로 업데이트되어 정사각형 공백을 유지합니다.
    • unlocked: width와 height를 개별적으로 설정해 직사각형 공백을 생성할 수 있습니다.
